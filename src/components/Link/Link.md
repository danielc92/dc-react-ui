### Features

- supports multiple sizes

### Small

```jsx
<Link size="small" href="https://github.com">
  Visit github
</Link>
```

### Default

```jsx
<Link size="medium" href="https://github.com">
  Visit github
</Link>
```

### Large

```jsx
<Link size="large" href="https://github.com">
  Visit github
</Link>
```
